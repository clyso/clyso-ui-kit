  # Installation

```bash
npm install @clyso/clyso-ui-kit
yarn add @clyso/clyso-ui-kit
```

To be able to use the library you need to have `vue` installed, and also `naive-ui`, `vue-router`, `vite-plugin-svg-icons`.

```bash
yarn add naive-ui vue-router
yarn add vite-plugin-svg-icons -D
```
